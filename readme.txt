
On The Fly Code Formatting Capabilities
by Lothar Mischke

Does on the fly (pretty print) code formatting in all code windows.

To Use/Load:
	Get Envy/QA from: http://www-3.ibm.com/software/ad/smalltalk/goodies/

	Import my config map

	Once loaded the filter can be switched on and off using a menu entry
	within the Transcript window.
